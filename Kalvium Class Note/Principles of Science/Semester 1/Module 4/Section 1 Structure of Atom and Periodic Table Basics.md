## Sequence of Matter Formation

Atoms ‚Üí Elements ‚Üí Molecules ‚Üí Compounds ‚Üí Macromolecules
- **Atom**: Smallest unit of matter.
- **Element**: Pure substance of one kind of atom (e.g., H, O).
- **Molecule**: Bonded atoms (e.g., H‚ÇÇO, O‚ÇÇ).
- **Compound**: Different atoms chemically bonded (e.g., NaCl, CO‚ÇÇ).
- **Macromolecule**: Large molecules (proteins, DNA).

## Key Terms

- **Orbit/Energy Level**: Region where electrons are found.
- **Atomic Structure**: Arrangement of p‚Å∫, n‚Å∞, e‚Åª.
- **Atomic Number (Z)**: Number of protons ‚Üí identity of element.
- **Mass Number (A)**: Protons + Neutrons.
- **Atomic Notation**:
    - Mass number (A) = superscript (top-left).
    - Atomic number (Z) = subscript (bottom-left).

Example:

- ‚Å∂‚ÇÉLi ‚Üí Lithium with A = 6, Z = 3.

## Periodic Table Basics

- **Purpose**: Organizes elements by atomic number ‚Üí shows repeating properties.
- **Structure**:
    - Rows ‚Üí **Periods** (increasing atomic number).
    - Columns ‚Üí **Groups** (similar chemical properties).
    - Special sections: **Transition metals, Lanthanides, Actinides**.

### Modern Periodic Table

- Based on **atomic number** (not mass).
- Solves isotope & ordering problems.
- Properties repeat periodically ‚Üí **Modern Periodic Law**.
- Similar to **concert seating** analogy:
    - Rows = arrival order (periods).
    - Sections = groups with similar properties.
    - VIP sections = unique blocks (f-block, transition metals).
