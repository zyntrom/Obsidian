## **Backend Blueprint – Clean Study Notes**

### **1. Magic Box Activity = How APIs Work**

- Mentor = “Server”
- Students = “Client/Frontend”
- “Magic words” = API endpoints
- Asking a question = making a request
- Server giving info = sending a response
- Asking something invalid = error response  
    **Core idea:** Frontend and backend communicate using structured requests and responses defined by an API.

---

## **2. What Is Backend?**

- Frontend = what the user sees (UI).
- Backend = the invisible engine handling:
    - Authentication
    - Storing/retrieving data
    - Search
    - Business logic
    - Security
- Backend = logic + database + performance + security + infrastructure.

---
